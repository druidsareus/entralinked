# Quick Reference\n\n## Configuration Methods\n\n### GUI Settings Tab (Easiest)\n\n1. Click **Settings** tab in main window\n2. Enter DNS address\n3. Click **Test Address** to validate\n4. Click **Save Settings**\n5. Restart application\n\n### Configuration File\n\nEdit `config.json`:\n\n```json\n{\n  \"dnsAddress\": \"192.168.1.100\"\n}\n```\n\nRestart application.\n\n### Command-Line\n\n```bash\njava -jar entralinked.jar --dns 192.168.1.100\n```\n\n## Examples\n\n### IPv4 Address\n\n```bash\njava -jar entralinked.jar --dns 192.168.1.100\n```\n\n### Hostname\n\n```bash\njava -jar entralinked.jar --dns game-server.local\n```\n\n### Without GUI\n\n```bash\njava -jar entralinked.jar disablegui --dns 192.168.1.100\n```\n\n### Auto-Detection (Default)\n\n```bash\njava -jar entralinked.jar\n```\n\n## Verification\n\nCheck the Console tab or logs for:\n\n```\nConfigure your DS to use the following DNS server: X.X.X.X\n```\n\n## Configuration Priority\n\n1. `--dns` flag (highest)\n2. `dnsAddress` in `config.json`\n3. `hostName` in `config.json`\n4. Auto-detect (default)\n"
